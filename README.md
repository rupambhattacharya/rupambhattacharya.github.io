# rupambhattacharya.github.io

Personal website of Rupam Bhattacharya — senior engineer in Munich working on AI, cloud, and platform engineering.

Live at <https://rupambhattacharya.github.io/>.

## Tech

Single-file static site. No build step, no framework, no JavaScript framework dependencies. Just HTML + CSS + Google Fonts. To preview locally:

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Editing

The whole site lives in `index.html`. Sections are clearly commented (Hero, About, Featured Work, Experience, Skills, Writing, Personal, Contact).

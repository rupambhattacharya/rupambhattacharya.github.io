Featured Work

Natural-Language Interface to OpenSearch

ProSieben · 2025–present · Internal platform

A stateful chat UI where product managers and engineers query the OpenSearch cluster in plain English. Claude translates intent into OpenSearch DSL queries via tool-use, executes them, and generates persistent dashboards from the results — cutting time-to-insight and freeing engineering from ad-hoc reporting requests.

Cross-stack engineering: schema-aware prompting against the OpenSearch index mapping, structured tool outputs, session state for follow-up queries, observability of LLM cost and latency, and a clean front-end. Now used by product and engineering daily.

Stack: Next.js · TypeScript · Anthropic Claude API (tool use) · OpenSearch

GenAI Developer Agent (CLI)

ProSieben · 2024–present · Internal platform

An agentic CLI assistant that orchestrates LLM calls (OpenAI and Anthropic) with tool integrations across Atlassian, GitLab, and Jira to automate multi-step developer workflows. Includes a multi-LLM evaluation pipeline that benchmarks model quality, cost, and latency across providers; prompts are shipped as versioned, deployable artefacts with a consistent release and rollback process.

The design treats internal AI as a product: centralised configuration, observable outputs, evaluations as a quality gate, and a clean abstraction that hides model complexity from end users.

Stack: TypeScript · Node.js · OpenAI & Anthropic APIs · function calling · GitLab + Jira APIs

AWS Platform Migration · Live TV AdTech

ProSieben · 2024–present

Led the end-to-end AWS platform migration for Live TV AdTech serving 25M+ daily users: consolidated a single shared account into isolated per-team accounts with Terraform-managed IaC, migrated container workloads from ECS EC2 to Fargate, integrated AWS Inspector into the build pipeline, and rebuilt CI/CD from scratch with GitLab CI.

Outcomes: 90% reduction in platform vulnerabilities, pipeline duration from 25 to 8 minutes (68% faster) while supporting 282 commits and 150 pipeline runs per day, and €8,000/month in infrastructure savings through rightsizing, Fargate Spot, Lambda optimisation, and decommissioning of legacy services.

Stack: AWS (Lambda, ECS Fargate, EKS, SQS/SNS, DynamoDB) · Terraform · GitLab CI · Datadog

Architected the shared content platform enabling self-service multi-brand publishing across 200+ publishers.
